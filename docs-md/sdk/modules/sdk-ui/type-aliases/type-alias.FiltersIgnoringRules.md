# Type alias FiltersIgnoringRules

> **FiltersIgnoringRules**: `object`

Filters ignoring rules.

## Type declaration

### `all`

**all**?: `boolean`

Boolean flag whether to ignore all filters.

If not specified, the default value is `false`.

***

### `ids`

**ids**?: `string`[]

Filter GUIDs to ignore.
