# Interface RadialGradientConfig

Radial gradient configuration.

## Example

```ts
const radial: RadialGradientConfig = {
  centerX: 0.5,
  centerY: 0.5,
  radius: 0.8
};
```

## Properties

### centerX

> **centerX**: `number`

X coordinate of the center point (0-1)

***

### centerY

> **centerY**: `number`

Y coordinate of the center point (0-1)

***

### radius

> **radius**: `number`

Radius of the gradient (0-1)
