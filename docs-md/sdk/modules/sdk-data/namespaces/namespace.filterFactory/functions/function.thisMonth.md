# Function thisMonth

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `dimension` | [`DateDimension`](../../../interfaces/interface.DateDimension.md) | date dimension to filter |

## Returns

[`Filter`](../../../interfaces/interface.Filter.md)
